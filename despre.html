Despre HTML

Taguri si elemente populare in HTML

Taguri utilizate in head:


Tagul <head> conține informații (metadate) care pot fi citite de mașină despre document, cum ar fi titlul, scripturile și foile de stil.

Tagul <title> definește titlul documentului care este afișat în bara de titlu a browserului sau în fila unei pagini. Conține doar text; etichetele din cadrul elementului sunt ignorate.

Tagul <link /> specifică relațiile dintre documentul curent și o resursă externă. Acest element este cel mai frecvent folosit pentru a lega foile de stil, dar este, de asemenea, folosit pentru a stabili pictograme de site, printre altele.

Elementul HTML <meta> reprezintă metadate care nu pot fi reprezentate de alte elemente HTML legate de meta.

Taguri utilizate in body:


Tagul <body>reprezintă conținutul unui document HTML. Într-un document poate exista un singur element <body>.

Detalii despre tagul <p>

  • Unul dintre cele mai folosite taguri.
  • Este folosit pentru paragrafe

Cel mai folosit element este P. P reprezinta un paragraf.

Un alt element popular este headingul. Headingurile pot fi de mai multe tipuri. Ele se diferentiaza prin numere.

Headingurile (<h>) sunt de mai multe tipuri:

Cel mai mare heading si cel mai important este h1, si cel mai mic este h6.

Spanul este un subelement, folosit pentru a selecta si manipula un cuvant o propotiztie sau o fraza.

Tagul <a> creează un hyperlink către pagini web, fișiere, adrese de e-mail, locații din aceeași pagină sau orice altceva poate adresa o adresă URL.

Tagul <section> definește o secțiune dintr-un document.

Tagul <div> este containerul generic pentru conținutul de flux.

Tagul <br /> inserează o singură întrerupere de linie.

Tagul <hr /> reprezintă o pauză tematică între elementele la nivel de paragraf: de exemplu, o schimbare de scenă într-o poveste sau o schimbare de subiect în cadrul unei secțiuni.

Tagul <ul> reprezintă o listă neordonată de articole, de obicei redată ca o listă cu marcatori.

Tagul <li> este folosit pentru a reprezenta un element dintr-o listă.

Tagul <img> încorporează o imagine în document.
